快速开始（纯国产模型，无需翻墙）

第一步：获取国内模型 API Key

至少注册一个国内平台，获取 API Key：

# 阿里云百炼（推荐，新用户100万token免费）
# 注册：https://dashscope.aliyun.com/
DASHSCOPE_API_KEY=sk-xxxxxxxxxxxx

# DeepSeek（超便宜，充$2用很久）
# 注册：https://platform.deepseek.com/
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxx

# 智谱AI（清华系，glm-4-flash超便宜）
# 注册：https://open.bigmodel.cn/
ZHIPU_API_KEY=xxxxxxxxxxxx.xxxxxxxx

# Kimi（月之暗面，长文本能力强）
# 注册：https://platform.moonshot.cn/
MOONSHOT_API_KEY=sk-xxxxxxxxxxxx

第二步：创建配置文件

# 复制模板并填入你的API Key
copy .env.example .env
notepad .env

第三步：安装依赖并启动

pip install -r requirements.txt
python main.py

然后打开浏览器访问 http://localhost:8000

第四步：上传任务，开始评估！

在网页上上传您的任务JSON文件，或使用示例任务
